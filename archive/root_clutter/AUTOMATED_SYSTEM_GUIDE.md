## 🎯 **HOW IT WORKS**

### **1. Start System:**
```
Double-click: START_FULLY_AUTOMATED_TRADING.bat
```

### **2. What Happens:**
1. **Background Process Starts** (minimized window):
   - Fetches data from Angel One API
   - Processes option chains
   - Generates trade signals
   - Executes paper trades
   - Updates PnL
   - Exports all files

2. **Profit Monitor Opens** (what you see):
   - Shows only: PnL, Trades, Profit
   - Auto-refreshes every 5 seconds
   - Clean, focused display

### **3. You Just Watch:**
- See your profit grow (or shrink)
- Monitor your trades
- Track win rate
- View open positions
