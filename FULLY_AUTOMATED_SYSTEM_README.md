## 🎯 **HOW IT WORKS**

### **Step 1: Start**
```
Double-click: START_FULLY_AUTOMATED_TRADING.bat
```

### **Step 2: What Happens**
1. **Background Process Starts** (minimized):
   - Connects to Angel One API
   - Fetches option chain data every 5 seconds
   - Processes all 5 indices (NIFTY, BANKNIFTY, FINNIFTY, MIDCPNIFTY, SENSEX)
   - Generates trade signals
   - Executes paper trades automatically
   - Updates PnL in real-time
   - Saves all data to files

2. **Profit Monitor Opens** (what you see):
   - Shows only: PnL, Trades, Profit
   - Auto-refreshes every 5 seconds
   - Clean, focused display
   - No technical details
